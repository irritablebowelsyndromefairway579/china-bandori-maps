# 全国邦群地图

一个以粉色 Material Design 3 风格为主题的全国邦群可视化地图页面。

用于聚合展示全国各省及海外地区的 BanG Dream! 群聊信息，支持地图缩放、拖拽、分省查看、国外切换与一键复制群号，帮助同好快速找到组织

## 项目说明

本项目是**纯静态项目**：

- 无后端服务代码
- 无构建步骤（不依赖打包器）
- 直接通过浏览器打开即可运行

## 主要功能

- 中国地图全屏展示，支持缩放与拖拽
- 省份群数量可视化（地图徽标）
- 右侧查看省份群详情
- 支持海外数据切换
- 群号一键复制
- 反馈二维码弹窗与外链跳转
- 移动端适配（含交互优化）

## 数据来源

- 在线接口：`https://mapapi.enldm.cyou/api/bandori`
- 本地兜底：`./bandori.json`

## 目录结构

- `index.html`：页面结构
- `styles.css`：样式与动画
- `app.js`：交互逻辑
- `china.js`：地图数据/绘制依赖
- `bandori.json`：本地数据兜底

## 运行方式

### 方式 1：直接打开

双击 `index.html` 用浏览器打开。

### 方式 2：本地静态服务器（推荐）

可使用任意静态服务器工具在项目目录启动后访问页面。
